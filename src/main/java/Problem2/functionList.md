### 2. FindAndRemoveDuplicate
- **LinkedList**로 첫 문자부터 순회
- 반복문자 나올 시 next  node **포인터 변경** (삭제)
- 반복이 없다면 다음 node로 이동
- ```return 최종 문자열```
